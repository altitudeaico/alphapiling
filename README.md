## What it demonstrates

This POC simulates the full automation workflow:

1. **📧 Email Scanning** - Monitors inbox for new tender enquiries
2. **📑 Document Classification** - AI identifies relevant pile drawings from 147+ pages
3. **⚙️ Data Extraction** - Extracts pile schedules (including separate load tables)
4. **📊 Excel Generation** - Outputs structured spreadsheet with all pile data
5. **📁 Drive Organisation** - Creates project folders and sorts attachments
6. **📋 Tracker Update** - Adds new row to tender tracking spreadsheet
7. **✉️ Auto-Reply** - Sends acknowledgement email to sender
